# Doce Digital 🍰

## Tema
Livro digital de receitas de confeitaria com sistema de utilizadores e gestão de conteúdo.

## Tecnologias
- PHP 8.3 (OOP)
- MySQL
- HTML5 + CSS3
- Bootstrap 5
- JavaScript

## Funcionalidades
- Registo e login de utilizadores
- Sessões e proteção de páginas privadas
- CRUD completo de receitas
- Sistema de comentários por receita
- Área de administração
- Pesquisa de receitas

## Base de Dados
4 tabelas: `utilizadores`, `receitas`, `ingredientes`, `comentarios`

## Como instalar
1. Instalar MAMP
2. Colocar o projeto em `htdocs/Simpa_Baker`
3. Importar o SQL no phpMyAdmin
4. Aceder a `http://localhost:8888/Simpa_Baker`

## Limitações
- Upload de imagens não implementado
- Sem paginação nas receitas
